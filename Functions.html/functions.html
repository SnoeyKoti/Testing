Understanding Functions in Programming

What is a Function?

A function is a block of code designed to perform a particular task. Functions allow code to be reused and organized more effectively.

Function Call

Calling a function is the process of executing the function. This is done by using the function name followed by parentheses, optionally including arguments.

            
            // Example in JavaScript
            function greet(name) {
                return "Hello, " + name + "!";
            }

            let message = greet("Alice"); // Function call
            console.log(message); // Outputs: Hello, Alice!

Function Definition

Defining a function involves specifying the function name, parameters, and the block of code that performs the task.

            
            // Example in JavaScript
            function add(a, b) {
                return a + b;
            }

Variables

Variables are used to store data values. They can be defined using keywords like var, let, or const in JavaScript.

            
            // Example in JavaScript
            let x = 5;
            const y = 10;
            var z = x + y; // z is 15

Data Types

Data types specify the type of data that can be stored and manipulated within a program. Common data types include:

  • String: Textual data, e.g., "Hello, World!"
  • Number: Numeric data, e.g., 42 or 3.14
  • Boolean: Logical data, e.g., true or false
  • Array: A collection of values, e.g., [1, 2, 3]
  • Object: A collection of key-value pairs, e.g., {name: "Alice", age: 30}
